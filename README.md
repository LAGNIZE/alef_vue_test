# alef_vue_test
Веб приложение разработано с помощью фреймворка VueJS. Пользователь может заполнить информацию о себе и своих детях.
Изначально пользователь видит только кнопку "+", при нажатии на нее появляется блок в котором можно ввести информацию о ребенке: Имя и возраст. Таким образом пользователь может добавить вплоть до 5 детей. Когда пользователь добавил 5 детей — кнопка "+" исчезает и больше недоступна. Так же напротив каждого ребенка есть кнопка "удалить" , при нажатии на которую соответствующая запись удаляется.
 
При нажатии на кнопку "Сохранить" введенные данные отображаются на странице "Превью".
